# Stress Type Quest

A game-styled personality quiz prototype built with React + TypeScript.

## Run locally

1. Install Node.js (if needed).
2. Install dependencies:
   - `npm install`
3. Start dev server:
   - `npm run dev`

## Current features

- Landing screen with 3-axis model
- Quiz flow with progress bar
- 22 questions total (Cognitive + Emotional + Social)
- 3-letter result code generation (`C -> E -> S`)
- Archetype title/blurb generation
- Share text copy button
